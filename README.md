# 📊 Superstore Sales & Profit Analysis Dashboard

## 📌 Project Overview
This project analyzes sales and profitability trends using the Superstore dataset. The interactive Tableau dashboard helps identify high-performing regions, profitable product categories, and business opportunities through data-driven insights.

---

## 🎯 Business Objectives
- Analyze sales performance across different regions.
- Identify the most and least profitable product categories.
- Understand profit trends to support business decision-making.
- Build an interactive dashboard for business stakeholders.

---

## 🛠️ Tools & Technologies
- SQL
- Tableau Public
- Git & GitHub
- CSV Dataset (Sample Superstore)

## 📈 Dashboard Features
- Sales by Region Analysis
- Profit by Sub-Category Analysis
- Interactive Filters and Drill-downs
- Color-coded Profit Visualization

---

## 📊 Key Insights
- ✅ West region generated the highest sales and profit.
- ✅ East region was the second-best performing region.
- ✅ Accessories and Paper were among the most profitable sub-categories.
- ✅ Some categories contributed significantly to sales but generated lower profits.
